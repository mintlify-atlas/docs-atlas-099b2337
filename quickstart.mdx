# Quick Start

This guide will help you implement your first date picker in just a few minutes.

## Choose Your Pattern

React Native Date Picker supports two main usage patterns:

<CardGroup cols={2}>
  <Card title="Modal Pattern" icon="window-restore">
    A popup modal with confirm/cancel buttons
  </Card>
  <Card title="Inline Pattern" icon="calendar">
    Embedded directly in your view
  </Card>
</CardGroup>

## Modal Pattern (Recommended)

The modal pattern is the most common way to use the date picker. It provides a clean popup experience with confirm and cancel actions.

<Steps>

<Step title="Import the component">
  ```jsx
  import React, { useState } from 'react'
  import { Button } from 'react-native'
  import DatePicker from 'react-native-date-picker'
  ```
</Step>

<Step title="Set up state">
  Create state variables to manage the date and modal visibility:
  
  ```jsx
  const [date, setDate] = useState(new Date())
  const [open, setOpen] = useState(false)
  ```
</Step>

<Step title="Render the DatePicker">
  Add the DatePicker component with modal props:
  
  ```jsx
  return (
    <>
      <Button title="Open" onPress={() => setOpen(true)} />
      <DatePicker
        modal
        open={open}
        date={date}
        onConfirm={(date) => {
          setOpen(false)
          setDate(date)
        }}
        onCancel={() => {
          setOpen(false)
        }}
      />
    </>
  )
  ```
</Step>

</Steps>

### Complete Modal Example

<CodeGroup>

```jsx App.js
import React, { useState } from 'react'
import { Button } from 'react-native'
import DatePicker from 'react-native-date-picker'

export default () => {
  const [date, setDate] = useState(new Date())
  const [open, setOpen] = useState(false)

  return (
    <>
      <Button title="Open" onPress={() => setOpen(true)} />
      <DatePicker
        modal
        open={open}
        date={date}
        onConfirm={(date) => {
          setOpen(false)
          setDate(date)
        }}
        onCancel={() => {
          setOpen(false)
        }}
      />
    </>
  )
}
```

```tsx App.tsx (TypeScript)
import React, { useState } from 'react'
import { Button } from 'react-native'
import DatePicker from 'react-native-date-picker'

export default () => {
  const [date, setDate] = useState<Date>(new Date())
  const [open, setOpen] = useState<boolean>(false)

  return (
    <>
      <Button title="Open" onPress={() => setOpen(true)} />
      <DatePicker
        modal
        open={open}
        date={date}
        onConfirm={(date: Date) => {
          setOpen(false)
          setDate(date)
        }}
        onCancel={() => {
          setOpen(false)
        }}
      />
    </>
  )
}
```

</CodeGroup>

<Tip>
  The modal pattern handles the UI presentation for you, including the backdrop and animations.
</Tip>

## Inline Pattern

The inline pattern embeds the picker directly in your view. This is useful when you want the picker to be always visible or integrate it into a custom layout.

<Steps>

<Step title="Import the component">
  ```jsx
  import React, { useState } from 'react'
  import DatePicker from 'react-native-date-picker'
  ```
</Step>

<Step title="Set up state">
  Create a state variable to manage the selected date:
  
  ```jsx
  const [date, setDate] = useState(new Date())
  ```
</Step>

<Step title="Render the DatePicker">
  Add the DatePicker component without the modal prop:
  
  ```jsx
  return <DatePicker date={date} onDateChange={setDate} />
  ```
</Step>

</Steps>

### Complete Inline Example

<CodeGroup>

```jsx App.js
import React, { useState } from 'react'
import DatePicker from 'react-native-date-picker'

export default () => {
  const [date, setDate] = useState(new Date())

  return <DatePicker date={date} onDateChange={setDate} />
}
```

```tsx App.tsx (TypeScript)
import React, { useState } from 'react'
import DatePicker from 'react-native-date-picker'

export default () => {
  const [date, setDate] = useState<Date>(new Date())

  return <DatePicker date={date} onDateChange={setDate} />
}
```

</CodeGroup>

<Note>
  With the inline pattern, you're responsible for the layout and positioning of the picker.
</Note>

## Selecting Different Modes

The picker supports three modes: `datetime`, `date`, and `time`.

<CodeGroup>

```jsx DateTime (Default)
<DatePicker
  date={date}
  onDateChange={setDate}
  mode="datetime"
/>
```

```jsx Date Only
<DatePicker
  date={date}
  onDateChange={setDate}
  mode="date"
/>
```

```jsx Time Only
<DatePicker
  date={date}
  onDateChange={setDate}
  mode="time"
/>
```

</CodeGroup>

## Common Customizations

Here are some common ways to customize the date picker:

### Setting Date Limits

Restrict the selectable date range:

```jsx
<DatePicker
  date={date}
  onDateChange={setDate}
  minimumDate={new Date('2020-01-01')}
  maximumDate={new Date('2025-12-31')}
/>
```

### Customizing Modal Appearance

Change the modal text and colors:

```jsx
<DatePicker
  modal
  open={open}
  date={date}
  onConfirm={(date) => {
    setOpen(false)
    setDate(date)
  }}
  onCancel={() => setOpen(false)}
  title="Select Date"
  confirmText="Done"
  cancelText="Close"
  theme="dark"
/>
```

### Setting Locale

Change the language and date format:

```jsx
<DatePicker
  date={date}
  onDateChange={setDate}
  locale="fr" // French
/>
```

<Info>
  The locale affects the date order, language, and AM/PM preferences. Use a [Locale ID](https://developer.apple.com/library/content/documentation/MacOSX/Conceptual/BPInternational/LanguageandLocaleIDs/LanguageandLocaleIDs.html) format.
</Info>

### Minute Intervals

Set the interval for minute selection:

```jsx
<DatePicker
  date={date}
  onDateChange={setDate}
  mode="time"
  minuteInterval={15}
/>
```

Available intervals: `1`, `2`, `3`, `4`, `5`, `6`, `10`, `12`, `15`, `20`, `30`

## Full-Featured Example

Here's a complete example with common customizations:

```jsx
import React, { useState } from 'react'
import { View, Button, Text, StyleSheet } from 'react-native'
import DatePicker from 'react-native-date-picker'

export default () => {
  const [date, setDate] = useState(new Date())
  const [open, setOpen] = useState(false)

  return (
    <View style={styles.container}>
      <Text style={styles.text}>Selected: {date.toLocaleString()}</Text>
      
      <Button title="Pick a date" onPress={() => setOpen(true)} />
      
      <DatePicker
        modal
        open={open}
        date={date}
        mode="datetime"
        minimumDate={new Date()}
        minuteInterval={15}
        title="Select Date and Time"
        confirmText="Confirm"
        cancelText="Cancel"
        theme="auto"
        onConfirm={(date) => {
          setOpen(false)
          setDate(date)
        }}
        onCancel={() => {
          setOpen(false)
        }}
      />
    </View>
  )
}

const styles = StyleSheet.create({
  container: {
    flex: 1,
    justifyContent: 'center',
    alignItems: 'center',
    padding: 20,
  },
  text: {
    fontSize: 16,
    marginBottom: 20,
  },
})
```
